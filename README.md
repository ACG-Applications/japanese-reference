# 🇯🇵 日本語リファレンス - Japanese Reference Sheets

A comprehensive Japanese reference tool with furigana, TTS, and quiz mode. Works offline as a PWA!

## 📚 Features

- **11 Reference Sheets**: Counters, Frequency, Numbers, Calendar, Months/Weeks/Years, Time, Duration/Location/Quantity, Directions & Locations, Classroom Vocabulary, Classroom Expressions
- **Furigana Toggle**: Show/hide reading guides
- **Translation Toggle**: Show/hide English translations
- **TTS (Text-to-Speech)**: Hear Japanese pronunciation
- **Quiz Mode**: Fill in the blank practice
- **PWA Support**: Install on your phone or desktop
- **Keyboard Shortcuts**:
  - `Alt+F`: Toggle furigana
  - `Alt+R`: Toggle translations
  - `Alt+T`: Toggle TTS
  - `Alt+Q`: Toggle quiz mode

## 🚀 Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. Start learning!

## 📱 Install as PWA

- **Chrome/Edge**: Click the install icon in the address bar
- **Safari (iOS)**: Tap Share → Add to Home Screen
- **Firefox**: Not supported (use Chrome/Edge)
