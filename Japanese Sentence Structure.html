Cards 1–2: Identification & Transitive Actions
#1 X は Y です / だ Identification
Meaning: X is Y  |  Particle: は (wa) = topic
POLITE (です)
PLAIN (だ)
AFF: [X] は [Y] です
AFF: [X] は [Y] だ
PAST: [X] は [Y] でした
PAST: [X] は [Y] だった
NEG: [X] は [Y] じゃないです
NEG: [X] は [Y] じゃない
P-NEG: [X] は [Y] じゃなかったです
P-NEG: [X] は [Y] じゃなかった
私 (わたし) は 学生 (がくせい) です。
→ I am a student.
💡 です/だ attach to NOUNS and NA-ADJECTIVES only.
For i‑adjectives, NO だ (e.g., 美味しい, not 美味しいだ).
#2 X は Y を [V]ます / [V]る Transitive Action
Meaning: X does [verb] to Y  |  Particle: を (o) = direct object
POLITE (ます)
PLAIN (る)
AFF: [X] は [Y] を [V]ます
AFF: [X] は [Y] を [V]る
PAST: [X] は [Y] を [V]ました
PAST: [X] は [Y] を [V]た
NEG: [X] は [Y] を [V]ません
NEG: [X] は [Y] を [V]ない
P-NEG: [X] は [Y] を [V]ませんでした
P-NEG: [X] は [Y] を [V]なかった
私 (わたし) は 本 (ほん) を 読 (よ) みます。
→ I read a book.
💡 を ONLY marks direct object of transitive verbs.
Plain past: 買う→買った, 食べる→食べた, する→した, 来る→来た(きた).

Cards 3–4: Existence & Liking
#3 [場所] に [もの] が あります / います Existence
Meaning: Y exists at X / There is Y at X  |  Particles: に = location, が = subject
POLITE
PLAIN
AFF: [場所] に [もの] があります / います
AFF: [場所] に [もの] がある / いる
PAST: [場所] に [もの] がありました / いました
PAST: [場所] に [もの] があった / いた
NEG: [場所] に [もの] がありません / いません
NEG: [場所] に [もの] がない / いない
P-NEG: [場所] に [もの] がありませんでした / いませんでした
P-NEG: [場所] に [もの] がなかった / いなかった
机 (つくえ) の 上 (うえ) に 本 (ほん) があります。
→ There is a book on the desk.
💡 ある = inanimate (books, chairs), いる = animate (people, animals).
Word order can flip: [もの] が [場所] に あります (emphasizes the thing).
#4 X は Y が 好き (すき) です / だ Liking
Meaning: X likes Y  |  Particle: が = emotion object
POLITE
PLAIN
AFF: [X] は [Y] が 好きです
AFF: [X] は [Y] が 好きだ
PAST: [X] は [Y] が 好きでした
PAST: [X] は [Y] が 好きだった
NEG: [X] は [Y] が 好きじゃないです
NEG: [X] は [Y] が 好きじゃない
P-NEG: [X] は [Y] が 好きじゃなかったです
P-NEG: [X] は [Y] が 好きじゃなかった
私 (わたし) は 寿司 (すし) が 好きです。
→ I like sushi.
💡 好き is a NA‑ADJECTIVE, not a verb.
Dislike = 嫌い (きらい). “Really like” = 大好き (だいすき).
Other emotion words with が: 怖い (scared), 嬉しい (happy).

Cards 5–6: Ability & Movement
#5 X は Y が できます (出来) / できる Ability
Meaning: X can do Y / X is able to do Y  |  Particle: が = ability object
POLITE
PLAIN
AFF: [X] は [Y] が できます
AFF: [X] は [Y] が できる
PAST: [X] は [Y] が できました
PAST: [X] は [Y] が できた
NEG: [X] は [Y] が できません
NEG: [X] は [Y] が できない
P-NEG: [X] は [Y] が できませんでした
P-NEG: [X] は [Y] が できなかった
私 (わたし) は 日本語 (にほんご) が できます。
→ I can speak Japanese.
💡 できる is GROUP 3 (irregular) verb (する → できる).
Also used for “it’s possible”: 明日 (あした) は できますか？
Alternative: verb potential form (e.g., 読める = can read) — but ができる is simpler.
#6 [場所] へ / に 行きます / 来ます Movement
Meaning: Go to X / Come to X  |  Particles: へ = direction, に = destination
POLITE
PLAIN
AFF: [場所] へ/に 行きます / 来ます
AFF: [場所] へ/に 行く / 来る
PAST: [場所] へ/に 行きました / 来ました
PAST: [場所] へ/に 行った / 来た
NEG: [場所] へ/に 行きません / 来ません
NEG: [場所] へ/に 行かない / 来ない
P-NEG: [場所] へ/に 行きませんでした / 来ませんでした
P-NEG: [場所] へ/に 行かなかった / 来なかった
学校 (がっこう) へ 行 (い) きます。
→ I go to school.
💡 へ emphasizes DIRECTION; に emphasizes DESTINATION.
In casual speech, へ is often pronounced “え”.
“Come back” = 帰る (かえる). Time + に: 8時 (はちじ) に 行きます.

Cards 7–8: Location/Means & Giving
#7 [場所/手段] で [動作] を します / する Location / Means
Meaning: Do Y at X / Do Y by means of X  |  Particle: で = location of action OR means/tool
POLITE
PLAIN
AFF: [X] で [Y] を します
AFF: [X] で [Y] を する
PAST: [X] で [Y] を しました
PAST: [X] で [Y] を した
NEG: [X] で [Y] を しません
NEG: [X] で [Y] を しない
P-NEG: [X] で [Y] を しませんでした
P-NEG: [X] で [Y] を しなかった
公園 (こうえん) で 本 (ほん) を 読 (よ) みます。
→ I read a book at the park.
バス で 学校 (がっこう) へ 行 (い) きます。
→ I go to school by bus. (means)
💡 で is NOT for existence (that’s に).
Means/tools: ペンで書く = write with a pen.
Language: 日本語で話す = speak in Japanese.
#8 [Giver] は [Receiver] に [Thing] を あげます / あげる Giving
Meaning: Give Z to Y  |  Particles: に = receiver, を = thing given
POLITE
PLAIN
AFF: [X] は [Y] に [Z] を あげます
AFF: [X] は [Y] に [Z] を あげる
PAST: [X] は [Y] に [Z] を あげました
PAST: [X] は [Y] に [Z] を あげた
NEG: [X] は [Y] に [Z] を あげません
NEG: [X] は [Y] に [Z] を あげない
P-NEG: [X] は [Y] に [Z] を あげませんでした
P-NEG: [X] は [Y] に [Z] を あげなかった
私 (わたし) は 友達 (ともだち) に 本 (ほん) を あげます。
→ I give a book to my friend.
💡 あげる = giving OUTWARD (from speaker to others).
くれる = giving INWARD (someone gives TO speaker):
友達が 本を くれました = My friend gave me a book.
もらう = receiving: 友達に 本を もらいました = I received a book from my friend.

Cards 9–10: Wanting & Questions
#9 X は Y が 欲しい (ほしい) です / だ Wanting (Things)
Meaning: X wants Y  |  Particle: が = desire object
POLITE
PLAIN
AFF: [X] は [Y] が 欲しいです
AFF: [X] は [Y] が 欲しい
PAST: [X] は [Y] が 欲しかったです
PAST: [X] は [Y] が 欲しかった
NEG: [X] は [Y] が 欲しくないです
NEG: [X] は [Y] が 欲しくない
P-NEG: [X] は [Y] が 欲しくなかったです
P-NEG: [X] は [Y] が 欲しくなかった
私 (わたし) は コーヒー が 欲 (ほ) しいです。
→ I want coffee.
💡 欲しい is an I‑ADJECTIVE (conjugates like 高い).
ONLY for NOUNS (things you want to HAVE).
For wanting to DO something: use 〜たい (e.g., 行きたい = want to go).
Don’t say 欲しいです for actions — that’s a common mistake!
#10 [Statement] + か Question
Meaning: Question?  |  Particle: か = question marker
POLITE
PLAIN
AFF: [Statement] ですか
AFF: [Statement] か
PAST: [Statement] でしたか
PAST: [Statement] だったか
NEG: [Statement] じゃないですか
NEG: [Statement] じゃないか
P-NEG: [Statement] じゃなかったですか
P-NEG: [Statement] じゃなかったか
学生 (がくせい) です か？ → Are you a student?
もう 食 (た) べた か？ (plain) → Did you already eat?
💡 No question mark needed (but often used in casual text).
In speech, rising intonation does the job too.
WH‑questions: どこ ですか？ = Where is it?
何 (なに) を 食べますか？ = What will you eat?
Casual: か is sometimes dropped (もう 食べた？).
